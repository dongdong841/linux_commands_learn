# vim
linux下的一款非常强大的文件编辑器
# command format
> vim [arguments] [file ..]
## 参数
* -v vi模式（类似vi）
* -e ex模式（类似ex）
* -E 改进的ex模式
* -d diff模式（类似vimdiff）
* -R 只读模式
* -b 二进制模式
* -n 没有swap模式
* -o 打开多个文件，为每个文件创建一个窗口
* -p 打开多个文件，为每个文件创建一个标签页
* \+ 打开文件后光标停在最后一行
* +\<NUM\> 打开文件后光标停在第NUM行
* -h 显示帮助信息
* --version 显示vim版本信息（实例说明参见details目录下的version_para.md）
